Fiedler Scada Editor

Powerful technological screen SVG editor on your browser

View on GitHub Download .zip Download .tar.gz

Supervisory control and data acquisition (SCADA) is a system for remote monitoring and control that operates with coded signals over communication channels. Fiedler Corporation develops automatic controllers as well as software for creating and displaying on-line status of the technology.

The editor allows you to draw technological screens, add there interactive elements and controls, and more.
